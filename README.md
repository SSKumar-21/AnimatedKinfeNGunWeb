# 🎯 Project Overview

Knife N Gun is a frontend concept project created to practice and demonstrate:

- UI/UX Design
- Animation Design
- Interactive Storytelling
- Landing Page Development
- Modern Web Effects
- Visual Branding

This is **not a playable game**.

The project represents a fictional social deduction game through cinematic visuals, role introductions, animated transitions, and immersive design elements.

# 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/SSKumar-21/AnimatedKinfeNGunWeb.git
```

Navigate to the project:

```bash
cd AnimatedKinfeNGunWeb
```

Open:

```bash
index.html
```

or run using Live Server.
